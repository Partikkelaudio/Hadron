Hadron state description sheet

State title

grReedyFormantStepSync

Type of sound

Monosynth with a reed-like timbre, clear timbral variations on the expression controls.

Controllers

Expression1: Grain Pitch, controls formant brightness
Expression2: Amount of Random Formant stepsequencer (with host bpm sync). Grain Envelope, controls brightness
Expression3: Grain Duration, controls formant presence
Expression4: GrainPitch FM index, controls formant fullness and width. GrainPitch FM frequency, detune and modulate formant spectrum

Midi keyboard control

Note Number: Pitch
Velocity: Amp
Amp (normalized velocity): Controls output amplitude.
Interval (from previous midi note) : ___

Suggested source waveform

Slot1: Any sampled or single cycle waveform
Slot2: Any sampled or single cycle waveform
Slot3: Any sampled or single cycle waveform
Slot4: Any sampled or single cycle waveform

Waveform sources/slots used: 1,2,3,4

Pitch definition mode

Grain Rate
Octave: 0
e.g.
440Hz at note num 69 = octave 0
220Hz at note num 69 = octave -1
