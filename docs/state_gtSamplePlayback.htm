Hadron state description sheet

State title

gtSamplePlayback

Type of sound

A simple sample playback state, any midi note triggers playback of the source sounds with no transposition and no time modification.

Controllers

Expression1: Wave Mix X axis. (source 1 to 2, or source 4 to 3, depending on the value of expression2)
Expression2: Wave Mix Y axis. (source 1 to 4, or source 2 to 3, depending on the value of expression1)
Expression3: Grain Rate
Expression4: Grain Duration. The transfer curve of the expression control sets the grains to start at medium long, then increase duration until a very long duration is reached when the control is at 25%, after this the grain duration decrease to very short grains when he control is at 100%.

Midi keyboard control

Note Number: Not used
Velocity: Amp

Suggested source waveform

Slot1: Any sound file, not suited for single cycle source.
Slot2: Any sound file
Slot3: Any sound file
Slot4: Any sound file

Waveform sources/slots used: 1,2,3,4

Pitch definition mode

Play back source sound with no pitch modification
Octave: Not applicable
