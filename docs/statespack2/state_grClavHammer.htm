Hadron state description sheet

State title

grClavHammer

Type of sound

Metallic Clavinetto timbre based on granular pitch and rate FM

Controllers

Expression1: "Hammer", attack timbral sharpness
Expression2: "Animation", formant and FM modulation amount during sustain phase
Expression3: "Stereo Octaviation", stereo channel masking amount
Expression4: "Inharmonicity", FM frequency offset.
    Approaching harmonic FM ratios at 0.5 and 1.0, all other values give more or less inhramonic spectra.

Midi keyboard control

Note Number: Pitch
Velocity: Amp

Suggested source waveform

Slot1: Sine, other waveforms may be used for a more aggressive timbre
Slot2: Not used
Slot3: Not used
Slot4: Not used

Waveform sources/slots used: 1

Pitch definition mode

Grain Rate
Octave: 0
e.g.
440Hz at note num 69 = octave 0
220Hz at note num 69 = octave -1
