Hadron state description sheet

State title

grFmStepSync_GrainFm

Type of sound

Grain Rate FM. Using a step function for modulation frequency, so the frequency to modulator ratio will be in different integer relationships. A square wave LFO flips back and forth between different modulation ratios. High modulation index will create unstable pitch slides.

Controllers

Expression1: Grain Pitch
Expression2: Grain Rate FM Frequency Step Function (0.5,1,2,3,4,5,6,7,8).
    Steps coincide with fractional values of the expression control,
    so e.g. an expression value of 0.4 equals an FM ration of 1/4,
    while expression value 0.5 equals FM ratio 1/5.
    Due to internal Hadron artifacts, settings below 10% on this expression controller will lead to alternating random ratios.
Expression3: Grain duration and LFO/Step tempo.
    Combined mapping where grain duration changes linearly over the fader range,
    while the LFO frequency steps back and forth between 1 and 2.
Expression4: Grain Rate FM Index

Midi keyboard control

Note Number: Pitch
Velocity: Amp

Suggested source waveform

Slot1: Sine, other waveforms may be used for a more aggressive timbre
Slot2: Sine, other waveforms may be used for a more aggressive timbre
Slot3: Sine, other waveforms may be used for a more aggressive timbre
Slot4: Sine, other waveforms may be used for a more aggressive timbre

Waveform sources/slots used: 1,2,3,4

Pitch definition mode

Grain Rate
Octave: 0
e.g.
440Hz at note num 69 = octave 0
220Hz at note num 69 = octave -1
