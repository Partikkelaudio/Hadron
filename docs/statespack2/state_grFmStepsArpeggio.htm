Hadron state description sheet

State title

grFmStepsArpeggio

Type of sound

Granular Pitch FM. Using a step function for modulation frequency, so the frequency to modulator ratio will be in different integer relationships. This state also use a bpm synced LFO to move through (arpeggiate) different modulator ratios

Controllers

Expression1: Grain Pitch
Expression2: LFO/Arpeggiator amount, controlling FM ratio in a step function (0.5,1,2,3,4,5,6,7,8).
    As the LFO amount is increased it will span more and more different ratios,
    and move faster through each separate ratio.
Expression3: Grain Duration
Expression4: Grain Pitch FM Index

Midi keyboard control

Note Number: Pitch
Velocity: Amp

Suggested source waveform

Slot1: Sine, other waveforms may be used for a more aggressive timbre
Slot2: Sine, other waveforms may be used for a more aggressive timbre
Slot3: Sine, other waveforms may be used for a more aggressive timbre
Slot4: Sine, other waveforms may be used for a more aggressive timbre

Waveform sources/slots used: 1,2,3,4

Pitch definition mode

Grain Rate
Octave: 0
e.g.
440Hz at note num 69 = octave 0
220Hz at note num 69 = octave -1
