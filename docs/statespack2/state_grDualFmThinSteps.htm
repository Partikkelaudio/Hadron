Hadron state description sheet

State title

grDualFmThinSteps

Type of sound

Combined grain rate FM and grain pitch FM. Using a step function for modulation frequency, so the frequency to modulator ratio will be in different integer relationships. Grain rate mod index is kept withing range where pitch is stable.

Controllers

Expression1: Grain Pitch FM Frequency Step Function.
    (1/5, 2/5, 3/5, 4/5, 1/3, 2/3, 1/4, 1/2, 3/4)
Expression2: Grain Pitch FM Index
Expression3: Grain Rate FM Frequency Step Function.
    (1/5, 2/5, 3/5, 4/5, 1/3, 2/3, 1/4, 1/2, 3/4)
Expression4: Grain Rate FM Index

Midi keyboard control

Note Number: Pitch
Velocity: Amp

Suggested source waveform

Slot1: Sine, other waveforms may be used for a more aggressive timbre
Slot2: Not used
Slot3: Not used
Slot4: Not used

Waveform sources/slots used: 1

Pitch definition mode

Grain Rate
Octave: +2
e.g.
440Hz at note num 69 = octave 0
220Hz at note num 69 = octave -1
