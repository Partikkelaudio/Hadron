Hadron state description sheet

State title

grCartoonVocals

Type of sound

Vocalise synth with cartoon-ish formants, expression selects wowels.

Controllers

Expression1: Wowel select (Xfade)
Expression2: Voice select (Xfade from low to high)
Expression3: Vibrato amount
Expression4: Vibrato rate

Midi keyboard control

Note Number: Pitch
Velocity: Amp

Suggested source waveform

Slot1: formWave0.wav (any of the formWaveN sources)
Slot2: formWave0.wav (should be the same as source 1)
Slot3: formWave2.wav (any of the formWaveN sources)
Slot4: formWave2.wav (should be the same as source 3)

Waveform sources/slots used: 1,2,3,4

Pitch definition mode

Grain Rate
Octave: 0
e.g.
440Hz at note num 69 = octave 0
220Hz at note num 69 = octave -1
