Hadron state description sheet

State title

gtLivesampling1

Type of sound

Sample manipulation, designed to be used with live sampling but may be just as useful for maipulating stored samples too. Manual control of time pointer (for manual time stretch or "time stops"), pitch transposition via midi key.

Controllers

Expression1:
   Manual time pointer
Expression2:
   Time pointer LFO and random deviation
      0% to 16%: Increase Random deviation a little
      16% to 33%: Increase LFO amount a little
      33% to 49%: Increase Random deviation to max
      49% to 66%: Increase LFO amount to max
      66% to 82%: Decrease Random deviation to none
      82% to 100%: Decrease LFO amount to none
Expression3:
    Grain Rate
Expression4:
    Wave Mix, crossfade sequentially through source 1,2,3 and 4

Midi keyboard control

Note Number: Transposition
Velocity: Amp, Envelope control (softer envelope on softer velocity)
Interval: Grain Duration modification

Suggested source waveform

Slot1: Any sampled waveform
Slot2: Any sampled waveform
Slot3: Any sampled waveform
Slot4: Any sampled waveform

Waveform sources/slots used: 1,2,3,4

Pitch definition mode

Grain Transpose
Octave: 0
Live sampled sources will play back untransposed at middle C.
Source waveforms (soundfiles) transposed so that note num 69 (middle A) should give a perceived fundamental of 440Hz
The original key stored with each source wave file may be octave displaced, in which case the same octaviation occurs on playback with this state.
