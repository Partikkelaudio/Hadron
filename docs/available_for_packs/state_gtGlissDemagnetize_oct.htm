Hadron state description sheet

State title

gtGlissDemagnetize_oct

Type of sound

Glisson synthesis with an alternating octave magnetization mask.
In glisson synthesis each grain has a pitch glissando. In this (gtGlissDemagnetize_oct) state, the ending pitch is above or below the central pitch, and the glissandi constantly pull away from the pitch center. This glissando pattern can be called a “demagnetize” pattern. The ending pitch for each grain alternates between an octave above and an octave below the central (start) pitch.

Controllers

Expression1: Grain rate decrease, creating longer sweeps
Expression2: Grain rate random deviation (and increase)
Expression3: Sweep shape (magnetization amount/spread)
Expression4: Manual time pointer (phase), effective for sampled waveforms

Midi keyboard control

Note Number: Transposition
Velocity: Amp
Interval: Grain Duration modification

Suggested source waveform

Slot1: Any waveform
Slot2: Not used
Slot3: Not used
Slot4: Not used

Waveform sources/slots used: 1

Pitch definition mode

Grain Transpose
Octave: 0
Live sampled sources will play back untransposed at middle C.
Source waveforms (soundfiles) transposed so that note num 69 (middle A) should give a perceived fundamental of 440Hz
The original key stored with each source wave file may be octave displaced, in which case the same octaviation occurs on playback with this state.
