Hadron state description sheet

State title

gtPitchTimeAndMix

Type of sound

Basic granular manipulation of samples, designed for live sampling but works just as well for sound files.

Controllers

Expression1: Grain Rate
Expression2: Manual Time pointer
Expression3: Wave Mix X axis. (source 1 to 2, or source 4 to 3, depending on the value of expression4)
Expression4: Wave Mix Y axis. (source 1 to 4, or source 2 to 3, depending on the value of expression3)

Midi keyboard control

Note Number: Transposition
Velocity: Amp
Interval (from previous midi note): Modify grain duration

Suggested source waveform

Slot1: Any live sampled segment or sound file
Slot2: Any live sampled segment or sound file
Slot3: Any live sampled segment or sound file
Slot4: Any live sampled segment or sound file

Waveform sources/slots used: 1,2,3,4

Pitch definition mode

Grain Transpose
Octave: 0
Source waveforms transposed so that note num 69 (middle A) should give a perceived fundamental of 440Hz
The original key stored with each source wave file may be octave displaced, in which case the same octaviation occurs on playback with this state.
