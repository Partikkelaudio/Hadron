Hadron state description sheet

State title

fxFallingRobos

Type of sound

Grain Delay Cloud with a falling pitch profile, randomized pitch and delay time.
A bit flangy sounding when expression 1&2 is at zero, more "bubbly" with other settings

Controllers

Expression1: Delay time randomization and offset. Grain duration decrease (with amp compensation)
Expression2: Grain pitch randomization and offset(fall). Grain rate increase
Expression3: Grain delay feedback
Expression4: Output routing modification of single grains (channel mask morphing). Grain rate increase

Midi keyboard control

Midi input not used for parameter control

Suggested source waveform

Slot1: audioInput
Slot2: audioInput
Slot3: audioInput
Slot4: audioInput

Waveform sources/slots used: 1,2,3,4

Mask tables

Channelmask:
maskA = default pan (stereo center)
maskB = sequence: Left-Right-Bandpass1-BP2-Lowpass-BP3-Delay1-Delay2

Pitch definition mode

Not applicable.
