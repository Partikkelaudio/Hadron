Hadron state description sheet

State title

fxSprayDelayPitch

Type of sound

Tempo synced Grain Delay with 4-voice pitching and grain spray.
Voice pitching is set to:
    Voice1: no transposition
    Voice2: up 3 semitones
    Voice3: up one octave
    Voice4: down 7 semitones

Controllers

Expression1:
    Grain Spray: delay time deviation, grain rate and channelmasking
        Channelmask morphing starts at 75%
Expression2:
    Time scaling 0 to 100% of:
        Voice1: half note
        Voice2: dotted quarter
        Voice3: quarter note
        Voice4: dotted 8th
Expression3:
    Grain Delay Feedback
    Delay effect feedback (active when Expression1 is at max)
Expression4:
    Wave(Voice) Mix
        at 0%: Voice1 only
        at 25%: Voice1&2
        at 50%: Voice2&3
        at 75%: Voice3&4
        at 100%: All

Midi keyboard control

Midi input not used for parameter control.

Suggested source waveform

Slot1: audioInput
Slot2: audioInput
Slot3: audioInput
Slot4: audioInput

Waveform sources/slots used: 1,2,3,4

Mask tables

Channelmask:
    A: Left, Right, (loop length=8 particles)
    B: Left, Right, BP1, BP2, LP, BP3, Delay L, Delay R (loop length=8 particles)

Pitch definition mode

Not applicable
