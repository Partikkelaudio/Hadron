Hadron state description sheet

State title

fxOctremolo

Type of sound

Granular tremolo effect with stereo pan and octave pitch shift.

Controllers

Expression1:
    Tremolo rate, bpm synced
        8ths notes (0% to 12.5%)
        dotted 16ths (12.5% to 25%)
        triplets (25% to 37.5%)
        16ths (37.5% to 50%)
        Quintuplets (50% to 62.5%)
        16th triplets (62.5% to 75%)
        32nd notes (75% to 87.5%)
        32nd triplets (87.5% to 99%)
        64ths notes (100%)
Expression2: Grain Duration (tremolo width)
Expression3: Stereo spread
Expression4: Downpitching of every second grain
    Pitching quantized to semitone transposition,
    except the first part of the control where fine tuning can be adjusted within one semitone range.

Midi keyboard control

Midi input not used for parameter control

Suggested source waveform

Slot1: audioInput
Slot2: not used
Slot3: not used
Slot4: not used

Waveform sources/slots used: 1

Pitch definition mode

Not applicable
