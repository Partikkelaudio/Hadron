Hadron state description sheet

State title

fxStoptime

Type of sound

Midi triggered “stop time” or “hold” effect. Audio will pass thru Hadron even when no midi notes are sent to it. When Hadron receives a midi note, it will hold the current audio signal until the midi note is released. Maximum hold time is approximately 12 seconds.

Controllers

Expression1: Time Stop Amount. Full stop when control is at zero, slow down time when the control is increased
Expression2: Time LFO Amount, a slowly modulating time pointer activated during time stops
Expression3: Time jitter, a random modulation of the time pointer during time stops
Expression4: Grain Duration, adjust to fit the type of audio input material. Longer durations will generally give a smoother time stop

Midi keyboard control

Any midi note input triggers the “stop time” processing.

Suggested source waveform

Slot1: audioInput
Slot2: Not used
Slot3: Not used
Slot4: Not used

Waveform sources/slots used: 1

Pitch definition mode

Not applicable
