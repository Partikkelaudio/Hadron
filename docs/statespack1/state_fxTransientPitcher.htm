State title

fxTransientPitcher

Type of sound

Pitch transpose of transients, leaving stable part of signal untransposed.

Controllers

Expression1: Pitch amount (down one octave to up one octave)
Expression2: Transient envelope release time
Expression3: GrainRate
Expression4: Feedback

Midi keyboard control

Note Number: Pitch transpose (general transpose factor), default (no transposition) at note number 60

Suggested source waveform

Slot1: audioInput
Slot2: not used
Slot3: not used
Slot4: not used

Waveform sources/slots used: 1

Pitch definition mode

Not applicable
