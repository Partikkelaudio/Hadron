State title

fxSyncDelayPitch

Type of sound

Tempo synced Grain Delay pattern with separate pitching of each delay tap (voice).

Controllers

Expression1:
    Pitch scaling, continuous from zero to:
        Voice1: no transposition
        Voice2: up 3 semitones
        Voice3: up one octave
        Voice4: down 7 semitones
Expression2:
    Time scaling (delay time):
        Voice1: half note
        Voice2: dotted quarter
        Voice3: quarter note
        Voice4: dotted 8th
    Note: the delay time is modified using a transfer function.
    The first 50% of the slider affects the delay time in continuous manner from 0 to 50% of the above.
    After this, it will proceed in steps (half note, triplet and whole note).
Expression3:
    Grain Delay Feedback
    Delay effect feedback     (be careful when Expression 1 is at zero at it will overdrive feedback)
Expression4:
    Wave(Voice) Mix
        at 0%: Voice1 only
        at 25%: Voice1&2
        at 50%: Voice2&3
        at 75%: Voice3&4
        at 100%: All

Midi keyboard control

Midi input not used for parameter control.

Suggested source waveform

Slot1: audioInput
Slot2: audioInput
Slot3: audioInput
Slot4: audioInput

Waveform sources/slots used: 1,2,3,4

Pitch definition mode
