Hadron state description sheet

State title

fxGrainFlanger

Type of sound

Granular flanger effect. All source waveforms should be set to the same source, and normally used with audioInput as the source.

Controllers

Expression1: Flanger Depth
Expression2: Flanger Rate
Expression3: GrainRate
Expression4: GrainDuration

Midi keyboard control

Midi input not used for parameter control, but the flanger LFO phase will be reset when a midi note is received. The LFO’s may come out of sync when another Hadron state is used, and a midi note will reset the LFO phases to the intended relationship for the GrainFlanger effect.

Suggested source waveform

Slot1: audioInput
Slot2: audioInput
Slot3: audioInput
Slot4: audioInput

Waveform sources/slots used: 1,2,3,4

Pitch definition mode

Not applicable
