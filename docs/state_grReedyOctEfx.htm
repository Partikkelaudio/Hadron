Hadron state description sheet

State title

grReedyOctEfx

Type of sound

Monosynth with a reed-like timbre, subtle timbral variations on the expression controls.

Controllers

Expression1: Grain Pitch, controls formant brightness
Expression2: Morph output routing (channelmasks). (Subtle) Grain Envelope, controls brightness
Expression3: Grain Duration, controls formant presence
Expression4: Grain Pitch FM index, controls formant fullness and width

Midi keyboard control

Note Number: Pitch
Velocity: Amp

Suggested source waveform

Slot1: Any sampled or single cycle waveform
Slot2: Any sampled or single cycle waveform
Slot3: Any sampled or single cycle waveform
Slot4: Any sampled or single cycle waveform

Waveform sources/slots used: 1,2,3,4

Mask tables

Channelmask:
maskA = stereo alternate L-R-L-R
maskB = sequence: Bandpass1-BP2-Lowpass-BP1-BP2-LP-BP3-LP
(Lowpass and Bandpass3 sends to Delay efx)

Pitch definition mode

Grain Rate
Octave: -1
e.g.
440Hz at note num 69 = octave 0
220Hz at note num 69 = octave -1
