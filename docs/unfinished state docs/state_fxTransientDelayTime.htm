fxTransientDelayTime

Transient dependent delay time, transients have short delay time, sustained sections of sound have long delay time.

Controllers

Expression2: Transient envelope release
Expression3: GrainRate
Expression4: Delay Feedback

Midi keyboard control

Midi input not used for parameter control.
