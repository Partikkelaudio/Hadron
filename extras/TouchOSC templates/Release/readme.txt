TouchOSC templates can either be transferred to the iPad/iPhone/iPod Touch over WiFi or through iTunes. Check out the links below to see how.

WiFi: http://hexler.net/docs/touchosc-configuration-layout-transfer-wifi
iTunes: http://hexler.net/docs/touchosc-configuration-layout-manage-itunes

Documentation for how to set up TouchOSC with MIDI can be found on the TouchOSC website:
http://hexler.net/docs/touchosc-setup-coremidi-network

For all TouchOSC related questions please consult the TouchOSC documentation: http://hexler.net/docs/touchosc

For information about the MIDI CC values used in the template see the bundled text file "CC-list - HadronT-OSC-multipage.txt"
